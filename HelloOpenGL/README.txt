There are command line arguments for this OpenGL program.
They include:
	-quad		Draws a single red quad with a black border.
	-circles	Draws two red circles with black borders.
	-mouse		Same as circles, but the two circles can be dragged on mouse click.
	-anim		Same as circles, but the circles move in a random direction and "screen-wrap".
	-crowd		Same as anim, but with 50 circles.
