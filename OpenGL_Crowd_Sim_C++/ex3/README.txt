Linux Instructions:
Install the following libraries: libsdl2-dev libsdl2-ttf-dev libglew-dev
Run make in the src/ directory where main.cpp resides

Run the compiled ex3 located in build/


Windows Instructions:
Compile with included .sln (requires visual studio 2017 community or better)

Change the include dir to reflect the location of the SDL2 library  and include folders on your computer.
